# HillsAgro Smart Farm

Smart Agriculture system for efficient cultivation in hilly regions.

## Features
- Farmer Dashboard
- Soil Moisture Monitoring
- Weather Information
- Crop Recommendation
- Irrigation Management
- Fertilizer Suggestion
- Hilly Region Farming Solutions

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Description
This system helps farmers in hill regions monitor field conditions and make better farming decisions using smart agriculture concepts.
